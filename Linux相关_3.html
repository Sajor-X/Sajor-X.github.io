首先下载jdk，jdk下载页面 卸载原有jdk 检查操作系统是否自带openjdk 输入java -version若出现java版本信息，表明操作系统目前自带的jdk，按以下方式进行卸载。 检查操作系统上自带的jdk rpm -qa|grep jdk 如果显示jdk的相关信息，则继续，否则跳过这一步， 卸载原有的jdk yum remove java* 检索所有的java套件 rpm -qa | grep java 如果显示： java-x.x.x-openjdk-x.x.x.xxx-x.x.x.x.el7.x86_64 … java-x.x.x-openjdk-hea

软件安装

首先到官网spark.apache.org下载spark，spark下载页面 安装 spark安装文件夹准备 在/usr/local/目录下创建名称为spark的文件夹： sudo mkdir -p /usr/local/spark 将下载好的spark安装包上传到服务器上。 之后解压到/usr/local/spark中 sudo tar -zxvf spark-2.4.5-bin-hadoop2.7.tgz 配置 编辑配置文件sudo vim /etc/profile 定位到最后一行，添加如下内容（版本注意修改） # spark export SPARK_HOME=/us
